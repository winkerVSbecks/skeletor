Utilities

Skeletor has a number of small utility classes that act as easy-to-use helpers. Sometimes it's better to use a utility class than create a whole new class just to float an element.

/* Utility Classes */

/* Make element full width */
.u-full-width

/* Make sure elements don't run outside containers (great for images in columns) */
.u-max-full-width

/* Float either direction */
.u-pull-right
.u-pull-left

/* Text alignment */
.u-text-center {
.u-text-left
.u-text-right

/* Clear a float */
.u-cf

/* Vertically center anything */
%u-vertical-align
