Typography

All type is set in rems so that font-sizes and spacial relationships can be responsively sized based on a single <html> font-size property. Out of the box, Skeletor never changes the <html> font-size, but the ability to do so is there in case you need it for your project. All measurements are still base 10 though so an <h1> with 5.0remfont-size just means 50px.
