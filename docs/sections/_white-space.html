White Space

A base .container class is available. It has a maximum width of 48em and is horizontally centred by setting the left and right margins to auto.

A 0.5rem square baseline grid has been selected for layout and alignment. White space can be adjusted using a simple set of utilities based on this grid.

Scale Margin Padding
0 .m0, .mt0, .mr0, .mb0, .ml0 .p0, .ph0, .pv0

Misc. .m-auto

m stands for margin and p for padding. You can specify the top, right, bottom & left margin by using the t, r, b, l suffixes. Padding can only be set symmetrically therefore, h refers to horizontal and v vertical. The numbers 0 - 4 indicate the scale.

This pattern is inspired by Basscss and Gravitons.
