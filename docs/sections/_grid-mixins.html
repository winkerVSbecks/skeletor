Grid Mixins

While the 12 Column Grid classes are available, it is recommended that you use the mixins instead. Use of pre‑calculated grid classes leads to the presemce of numerous non-semantic class names throughout a document such as row, column, etc. Using the mixin keeps the markup clean and allows you to use appropriate semantic tags such as main, section and aside instead.

/* Two column (blog style) layout */
main {
  @include row;
}
/* 8 Column div for the blog posts */
section {
  @include col(8);
}
/* 4 Column div for the side-bar */
aside {
  @include col(4);
}
