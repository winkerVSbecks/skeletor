CSS Stats

cssstats parses the compiled CSS stylesheet and returns an object with statistics. These are high-level stats which help you analyse the performance and maintainability of your code. It helps you avoid bloat, catch redundant rules, and keep a low average specificity of selectors.

The visualization of these stats are available at localhost:3000/css/stats. The visualization is based on CSS Stats by Mrmrs & Jxnblk.
