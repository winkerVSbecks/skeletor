Flexbox

The Flexbox Layout module provides a more efficient way to construct layouts. The core idea behind it is to allow the container to alter the width, height and order of it's child elements to maintain the expected layout. It can expand items to fill the available space or shrink them to prevent overflow.

Bourbon provides an exhaustive set of flexbox mixins. Beyond that Skeletor provides a few utility classes to handle some common use cases.

.flex { display: flex; }

.flex-column { flex-direction: column; }
.flex-wrap { flex-wrap: wrap; }

.flex-center { align-items: center; }
.flex-stretch { align-items: stretch; }

.flex-auto { flex: 1 1; }
.flex-grow { flex: 1; }
